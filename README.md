
## Budal an Experimental Java HTTP Server (Pure Java and without framework)

## Getting started

- Install `Maven`
https://maven.apache.org/download.cgi

- Build `jar`
```shell
$ mvn clean package
```

- Run `jar`
```shell
$ java -jar target/Budal-1.0-SNAPSHOT.jar
```

## Run Tests

```shell
$ mvn test
```
